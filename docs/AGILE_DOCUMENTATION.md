# GroupSync — Agile Documentation (V2)

## Sprint 2 — Kanban Board + Discussions
**Duration:** 2 weeks
**Sprint Goal:** Visual task workflow with drag-and-drop and team discussions.

### V2 User Stories
| ID    | User Story                                                                         | Points |
|-------|------------------------------------------------------------------------------------|--------|
| US-13 | As a student, I want to see tasks in Kanban columns so I can track workflow         | 5      |
| US-14 | As a student, I want to drag tasks between columns to update status quickly         | 5      |
| US-15 | As a student, I want to click a task to see full details in a side panel            | 5      |
| US-16 | As a student, I want to change status via clickable pills                           | 2      |
| US-17 | As a student, I want to post comments on tasks for team discussion                  | 5      |
| US-18 | As a student, I want to see who posted each comment and when                        | 3      |
| US-19 | As a student, I want to see comment count badges on task cards                      | 1      |
| US-20 | As a student, I want a mini progress bar showing completion                         | 2      |

### Sprint 2 Retrospective
**What went well:** Kanban board came together quickly, drag-and-drop works without external libraries, comments sync in real-time via Firestore.
**What didn't go well:** Drag-and-drop not ideal on mobile, task detail modal could be split into smaller components.
**V3 improvements:** Add deadline tracking, progress dashboard, per-member analytics.
